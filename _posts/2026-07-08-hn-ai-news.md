  - title: Reducing Doom Loops with Final Token Preference Optimization
    link: https://www.liquid.ai/blog/antidoom
    domain: liquid.ai
    summary: Liquid AI's 'Antidoom' surgically eliminates LLM repetition loops by training models to prefer coherent alternatives at the single token where a loop begins
    points: 29
    hn_url: https://news.ycombinator.com/item?id=48820127
    comments: 0
    time: Jul 7, 16:47 UTC
    content_bullets:
    - Doom loops form when overtrained discourse tokens (e.g. 'Wait', 'Alternatively') dominate under low-temperature sampling, causing each repetition to reinforce itself until the context window is exhausted.
    - Antidoom detects loops by finding spans that repeat 4+ times over 60+ chars, then pinpoints the exact first diverging token as the sole training target.
    - Final Token Preference Optimization (FTPO) adapts DPO to train only trailing tokens, accepts multiple 'chosen' alternatives per sample, and uses a KL-like logit-space loss to redistribute probability with minimal side effects.
    - On Qwen3.5-4B, doom-loop rate dropped from 22.9% to 1%; on LFM2.5-2.6B from 10.2% to 1.4%, with benchmark scores improving across all temperatures.
    - Post-training, near-greedy (low-temperature) sampling outperforms higher temperatures — inverting conventional wisdom for reasoning models; code and pipeline are open-sourced on GitHub.
    discussion_bullets:
    - A commenter notes doom loops in coding agents often stem from context pollution — failed fix attempts accumulating in context — and that clearing the session and restarting with a precise symptom description usually beats further correction attempts.
    - 'One developer shared a workaround: a shim that asks the model to summarize what didn''t work, rolls back context to before the failed attempts, and injects only the summary — noting it helps but would benefit from first-class RL support.'
    - Another commenter argues the root cause is RLHF post-training optimizing for confidence and optics rather than transparent honesty, framing doom loops as a symptom of that deeper misalignment.
