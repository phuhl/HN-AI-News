      and the op that produced it — the graph builds itself implicitly.
    - 'Backpropagation applies the chain rule: when a variable reaches the loss via multiple paths, gradient contributions
      from all paths accumulate via += rather than overwriting.'
    - Backward (reverse) mode requires just one sweep to compute gradients for all parameters simultaneously; forward mode
      needs one sweep per parameter — impractical at millions of params.
    - The backward pass first topologically sorts the graph so each node follows its dependencies, seeds the output gradient
      to 1, then traverses in reverse applying op-specific local derivative rules.
